**2a. Create FloatingCommandBar component:**

Create `src/features/panels/components/FloatingCommandBar.tsx`:

Props: `{ viewId: ViewId }`

**Collapsed state:**

- Fixed position: `fixed bottom-6 right-6 z-30`
- Circular button: `w-10 h-10 rounded-full bg-primary text-primary-foreground`
- Icon: LayoutDashboard (from lucide-react)
- When edit mode is active: add pulsing ring `animate-pulse ring-2 ring-primary/40`

**Behavior on click:**

- If NOT in edit mode: enter edit mode AND expand menu
- If in edit mode and menu collapsed: just expand menu
- If in edit mode and menu expanded: do nothing (user uses Done to exit)

**Expanded state:**

- Menu appears above the FAB, growing upward
- 3 items stacked vertically with 8px gap:
  1. "Done" button (exits edit mode + collapses) — only shown when in edit mode
  2. "Add Panel" button (opens AddPanelDialog)
  3. "Reset" button (resets layout to default)
- Each item: pill-shaped, `bg-card border border-border px-3 py-1.5`
- Labels: `font-mono text-[10px] uppercase tracking-[0.15em]`
- Icons: small (14px) left of label
- Staggered entrance animation: each item delayed 50ms after previous

**Click outside to close:**

- Add a backdrop div or use a click-outside hook
- Closing the menu does NOT exit edit mode
