**One-way data flow rule (from video ~10:00):**

- `shared` (src/components/shared/, src/hooks/, src/types/, src/lib/) → can be imported by anyone
- `features` (src/features/\*/) → import from shared only, NEVER from other features
- `app` (src/views/, src/router/, panel.registry.ts) → can import from features and shared
