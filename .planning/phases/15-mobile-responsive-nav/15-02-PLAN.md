Create the bottom tab bar component:

**BottomTabBar.types.ts:**

- No props needed (self-contained — reads route from React Router)

**BottomTabBar.tsx:**

- Use `useLocation()` from react-router to detect active route
- Use `useNavigate()` for tab clicks (or `<NavLink>` for automatic active state)
- 4 tabs: Dashboard (/dashboard), Fleet (/fleet), Map (/map), Pilot (/pilot)
- Icons: LayoutDashboard, Users, Map, Gamepad2 from lucide-react
- Fixed bottom: `fixed bottom-0 inset-x-0 z-30 h-14`
- Styling: `bg-card border-t border-border`
- Active: `text-primary`, Inactive: `text-muted-foreground`
- Labels: `font-mono text-[10px] uppercase tracking-[0.1em]`
- `aria-current="page"` on the active tab link
- `role="navigation"` with `aria-label="Bottom navigation"` on the nav element

Use `<NavLink>` from react-router (it auto-applies active class and aria-current):

```tsx
<NavLink
  to="/robot-telemetry-dashboard/dashboard"
  className={({ isActive }) =>
    cn(
      'flex flex-col items-center gap-0.5 px-3 py-1',
      isActive ? 'text-primary' : 'text-muted-foreground'
    )
  }
>
  <LayoutDashboard size={20} />
  <span className="text-[10px] font-mono uppercase tracking-[0.1em]">
    Dashboard
  </span>
</NavLink>
```

Component is self-contained — doesn't need a mobile check (parent handles visibility).
